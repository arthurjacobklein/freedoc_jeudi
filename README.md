# FreeDoc

## How to install?
```
# Clone the project
$ git clone https://github.com/arthurjacobklein/freedoc_jeudi.git

# CD into the folder
$ cd freedoc_jeudi

# Install gems
$ bundle install

# Launch the migrations
$ rails db:migrate

# Create seed
$ rails db:seed

OPTIONAL
# If you have a bug when you launch your console, type this before `rails console`
$ spring stop
```
